## ğŸ†˜ Troubleshooting

### Site not updating after changes?
- Wait 2-3 minutes for Netlify to rebuild
- Check Netlify dashboard for deploy status
- Clear your browser cache (Ctrl+Shift+R or Cmd+Shift+R)

### Images not showing?
- Make sure image files are in the `images/` folder
- Check that filenames match exactly (case-sensitive!)
- Verify image file extensions (.jpg not .jpeg)

### Broke something?
- GitHub saves every version of your files
- Click "History" on any file to see previous versions
- You can revert to any previous version
